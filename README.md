# energy-consumption-iot
# Mini-projet – Suivi et analyse de la consommation énergétique

## Description
Ce projet simule un système de suivi de la consommation énergétique basé sur des capteurs IoT.
Il repose sur la programmation orientée objet, la simulation de données, la détection d’anomalies
et le stockage dans MongoDB.

## Fonctionnalités
- Simulation de capteurs IoT
- Génération de données énergétiques aléatoires
- Détection d’anomalies
- Stockage des données dans MongoDB
- Tests unitaires avec Pytest
- Vérification du style avec flake8

## Technologies
- Python
- MongoDB
- Pytest
- flake8

```bash
python src/main.py
